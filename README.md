# 🎮 DuelBet — Платформа прогнозов на стримерские дуэли

## Как задеплоить на Vercel за 3 минуты

### Вариант 1: Через GitHub (рекомендуется)

1. Создай новый репозиторий на [github.com](https://github.com)
2. Загрузи все файлы из этой папки
3. Зайди на [vercel.com](https://vercel.com) → "Add New Project"
4. Выбери свой репозиторий
5. Настройки оставь по умолчанию (Vercel сам определит React)
6. Нажми **Deploy** → получи ссылку вида `duelbet.vercel.app`

### Вариант 2: Через Vercel CLI

```bash
npm install -g vercel
cd duelbet
npm install
vercel --prod
```

## Функции MVP

- 🔴 **Live матчи** — карточки с прогрессом ставок в реальном времени
- 🎯 **Прогнозы** — выбираешь участника + сумму + жмёшь "Поставить"
- 📺 **Стрим внутри** — встроенный плеер, не надо переходить
- ⚡ **Лента активности** — видно кто и сколько поставил
- 👑 **Панель организатора** — создаёшь матч → получаешь ссылку → объявляешь победителя
- 💰 **Баланс** — система кошелька (пока симуляция)

## Следующие шаги (бэкенд)

Для реальных денег понадобится:
- **База данных**: Supabase или PlanetScale (бесплатно для старта)
- **Авторизация**: Supabase Auth или NextAuth
- **Платежи**: ЮKassa для самозанятых (лицензия не нужна при доходе < 2.4M/год)
- **WebSockets**: Pusher или Supabase Realtime для live-обновлений

## Концепция монетизации

- 10% комиссия с призового фонда
- Платный статус организатора (500₽/месяц)
- Реклама на стримах
